## Português

### Sobre o Projeto
Esta Web **API RESTful** é a evolução do sistema de vendas desenvolvido anteriormente em console. O objetivo desta etapa foi migrar a lógica de negócios para um ambiente web, utilizando **ASP.NET Core** e **Entity Framework Core** para fornecer persistência de dados robusta e endpoints escaláveis.

### Diferenciais Técnicos
- **Entity Framework Core:** Uso de ORM para mapeamento objeto-relacional, facilitando a interação com o banco de dados.
- **Migrations:** Gerenciamento de histórico e evolução do esquema do banco de dados de forma automatizada.
- **RESTful API:** Implementação de métodos HTTP (GET, POST, PUT, DELETE) seguindo as melhores práticas.
- **Injeção de Dependência:** Uso do container nativo do .NET para desacoplamento de serviços e repositórios (Repository Pattern).
- **Documentação Interativa:** Configuração do **Scalar (OpenAPI)** para testes e visualização dos endpoints.

## English

### About the Project
This **RESTful Web API** represents the evolution of the previously developed console-based sales system. The goal of this stage was to migrate the business logic to a web environment, using **ASP.NET Core** and **Entity Framework Core** for robust data persistence and scalable endpoints.

### Technical Highlights
- **Entity Framework Core:** Utilizing ORM for object-relational mapping, streamlining database interactions.
- **Migrations:** Automated management of database schema history and evolution.
- **RESTful API:** Implementation of HTTP methods (GET, POST, PUT, DELETE) following industry best practices.
- **Dependency Injection:** Leveraging the native .NET container for decoupling services and the Repository Pattern.
- **Interactive Documentation:** Swagger (OpenAPI) integration for easy endpoint testing and visualization.
